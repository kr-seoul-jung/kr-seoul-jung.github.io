# 중구 · 서울특별시

> 중구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-jung.github.io/**

중구는 서울 도심 한복판에 위치한 자치구로, 명동과 남대문시장을 축으로 한 상업과 남산을 품은 도심 녹지가 어우러진 곳이다. 서울시청, 서울역, 을지로 업무지구가 한데 모여 대한민국 행정과 교통, 금융의 관문 역할을 오랫동안 해 왔다. 덕수궁 돌담길과 명동성당, 정동 일대에는 개항기와 근대의 흔적이 골목마다 촘촘히 남아 있다. 낮에는 넥타이 부대와 외국인 관광객으로, 밤에는 명동과 을지로 노포 상권으로 밤낮없이 붐비는 도심의 심장이다. 면적은 서울에서 가장 작지만 오가는 유동 인구와 경제적 밀도는 어느 구에도 뒤지지 않는다.
